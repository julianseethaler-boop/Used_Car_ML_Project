# Used Car Price Prediction

Dieses Projekt untersucht Gebrauchtwagendaten und entwickelt schrittweise
Machine-Learning-Modelle zur Vorhersage des Verkaufspreises.

## Projektstruktur
- `notebooks/` – Datenaufbereitung, EDA und Modellierung
- `data/processed/` – bereinigte Datensätze

## Workflow
1. Notebook 01: Datenbereinigung
2. Notebook 02: Explorative Datenanalyse & Feature Selection
3. Notebook 03: Lineare Regression und Regularisierung
4. Notebook 04: Gradient Boosting
